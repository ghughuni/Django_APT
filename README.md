# Amazon Price Tracker Web App
This project is a Django-based web application that allows users to track the prices of products on e-commerce websites. 

## Front-end:
- *HTML*
- *CSS*
- *Bootstrap*
- *JavaScript*
- *jQuery*

## Database:
- *SQLite*

## Backend:
- *Python* 
- *Django*
- *Beautiful Soup*

# Key Functions:
- **Add Product with URL**
- **Delete Product**
- **Update Products**
